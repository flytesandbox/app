# Archive

This directory stores sanitized support records and generated evidence that
should not live in active Wiki navigation.

## Sections

- `support-generated/` is the archive path that `Support` uses for sanitized
  reports, manifests, and evidence snapshots.

## Rules

- Archive content is reference material, not the current control plane.
- Active docs may link into the archive, but they must not turn archive records
  into hidden runtime or deployment rules.
- Secrets, private keys, full database URLs, host fingerprints, and private IPs
  remain redacted in archive content.
