# Wiki and Support Contract

The Wiki and `Support` are separate services with a narrow integration contract.

## Wiki Responsibilities

- hold current-state ecosystem documentation
- store sanitized archives
- stay static and non-executable

## Support Responsibilities

- observe and analyze approved ecosystem signals
- publish sanitized archive outputs
- avoid material changes to the system it monitors

## Shared Rule

- Active Wiki pages and active `Support` outputs must describe the current
  ecosystem, not the tooling that originally assembled it.
